Never run git add, git commit, git push, or any other git write command.
The user handles all git operations manually in their own terminal.

**Why:** User explicitly interrupted a git commit attempt and stated this preference.

**How to apply:** After finishing file edits for a task, stop. Tell the user the files are ready and remind them to commit manually. Do not stage, commit, or push anything.
