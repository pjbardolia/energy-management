# Energy Management SaaS Platform

## Purpose

Industrial IoT platform for textile processing factories.

Current Factory:
Shiv Shakti Prints Pvt Ltd

- Raspberry Pi 3B+
- RS485 Modbus RTU
- INVT CHF100A VFDs

- Docker
- Portainer
